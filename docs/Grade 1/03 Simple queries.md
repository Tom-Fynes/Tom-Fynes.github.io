# Simple Queries
 
### SELECT Statement
The `SELECT` statement is used to retrieve data from one or more tables. It is the most commonly used SQL command.


#### Select All Columns
```sql
SELECT *
FROM table_name;
```

#### Select Specific Columns
```sql
SELECT firstname, lastname
FROM employees;
```
